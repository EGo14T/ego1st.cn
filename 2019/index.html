年终总结

2019。2020，

发表于2020-1-1•文章字数1672•被
人看爆

2019年结束了，在这里总结一下2019年我做了些什么，以及立下2020年的Flag 😝

代码

算是后半年开始发力吧，因为就业的压力比较大，迫于提高自己的代码水平。

每天主要提交的代码是LeetCode上刷的题（目前刷到28道了）,还有昕音乐项目以及博客的提交。

在新的一年里，定下以下几个小目标😜

  1. 继续坚持刷题，目标100道，不求多，但是要学到其中算法的思想
  2. 昕音乐项目发布上线，并坚持维护
  3. 博客坚持写，把踩到坑，学到的东西，还有有趣的事情都记录下来

技能

2019年7月，参加了实习，在公司里视野开阔了许多，也学到了许多新东西。在这里感谢我的导师教会了我许多东西，让我得到了非常大的提高。

🖥️买了腾讯云的服务器，并借此机会学会了Linux的使用

:robot:Docker容器化服务的简单使用

🚀学会了Vue以及Electron

🤤机缘巧合，学会了编写油猴脚本（其实就是写JS）

🔭会使用Python的selenium库进行自动化测试，顺道学会了Node.js的Puppeteer，两个东西都是自动化操作浏览器

😀大学里没学会的Java多线程在一次需求开发中也掌握了

😎写业务代码的水平自认为也提高了许多，并向着写出优雅代码的目标不断努力

:thinking:不知道算不算技能，会用IOS的快捷指令工具写一些炫酷的指令

2020年，希望自己能够掌握更多的技能，技多不压身嘛😚

  1. Java基础更加牢固，会用一些新特性开发
  2. MySQL要更加熟练
  3. Linux的操作水平达到导师80%的程度:roll_eyes:
  4. 锻炼自己看源码的能力
  5. 是时候提高独立能力了😭

健康

自从参加工作以来，好像吃胖了一点，毕竟一坐坐一天☺️

上次连续加班后，突然心绞痛，去医院检查了心脏和肺，都没有问题，这让我意识到了，身体是革命的本钱，一定不能连续熬夜😵

希望自己在2020年，健健康康，身体倍儿棒，吃嘛嘛香，再长高5厘米😝
